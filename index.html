Connect Using WeeChat

Follow these simple steps to connect to the Brockman network using WeeChat:

/server add brockman brockman.news/6667 -autoconnect

If you already have an account with NickServ, you can log in securely using SASL PLAIN. Otherwise, you can skip this step. 

/set irc.server.hackint.sasl_mechanism PLAIN
/set irc.server.hackint.sasl_username <login>
/set irc.server.hackint.sasl_password <password>

Finally, connect.

/save
      /connect brockman
      /join #all

Connect Using irssi

Follow these simple steps to connect to the Brockman network using irssi. If you already have an account with NickServ, you can log in using SASL PLAIN (optional):

/network add -sasl_username <login> -sasl_password <password> -sasl_mechanism PLAIN brockman

Then set up the connection like this:

      /server add -auto -net brockman brockman.news 6667
      /save

Finally, connect.

      /connect brockman
      /join #all
